EXPRESSION

An expression is a combination of values, variables, operators, and function calls that evaluate to a single
        value. 
Expressions produce a value.
Expressions:
Evaluation: Expressions are evaluated to produce a value. 
Examples:
5 + 3 (arithmetic operation) 
x > y (comparison) 
myFunction(a, b) (function call) 
name (a variable) 
Usage: Expressions can be part of a statement, or they can be used independently to compute a value.

STATEMEMT

A statement, on the other hand, is a complete unit of execution that performs an action or instruction.
Statements perform actions
Statements:
Execution: Statements perform an action or instruction. 
Examples:
int x = 10; (variable declaration and initialization) 
if (x > 5) { ... } (conditional statement) 
for (int i = 0; i < 10; i++) { ... } (loop statement) 
return result; (return statement) 
Usage: Statements form the core logic of a program, controlling the flow of execution and performing operations.
